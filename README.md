# SURAKSHA – Intelligent Safety Pendant

## 📌 Overview
An IoT + AI based wearable safety device designed for women and vulnerable individuals.

## 🔧 Features
- GPS based live tracking
- GSM emergency SMS alerts
- AI weapon detection using YOLO
- Camera recording for evidence
- One-touch emergency activation

## ⚙️ Requirements

### Software
- Python 3.x
- OpenCV
- Flask
- Ultralytics YOLO

Install:
```bash
pip install -r requirements.txt
